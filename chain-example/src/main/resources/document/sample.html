NonChain 介绍

NonChain 是一个轻量级的 Java LLM 应用开发库。

核心特性

LLM Provider 抽象、Embedding 向量化、知识库存储与检索。

支持的格式

格式扩展名依赖库
文本.txt无
Markdown.mdcommonmark
PDF.pdfPDFBox
Word.docxApache POI
HTML.htmljsoup

代码示例


DocumentReaders readers = new DocumentReaders()
    .register(new TxtDocumentReader());
ParsedDocument doc = readers.read(new File("example.txt"));
